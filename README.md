# 배당금 프로젝트
## 주요 기능
 * 회사 이름을 Input으로 받아서 해당 회사의 메타정보와 배당금 정보를 반환
 * 자동완성 기능(prefix)
 * 서비스에서 관리하고 있는 모든 회사 목록을 반환
 * 새로운 회사 정보 추가
 * 회사정보 삭제(+ 배당금 정보 / 캐시)
 * 회원가입 API
 * 로그인API
 
## 사용 기술 스택
 * JDK 11
 * Java
 * redis
 * Spring-boot
 * h2
 * JWT
 * spring security
 * lombok
 * Jsoup
 * Spring Data Jpa
